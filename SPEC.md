# TV Girl Memory Website Specification

## Project Overview
- **Project Name:** Lo-Fi Love - TV Girl Memory Website
- **Type:** Single-page romantic photo memory website
- **Core Functionality:** Display 6 photos with cute notes in a vintage TV Girl aesthetic
- **Target Users:** A couple wanting to showcase their memories

## UI/UX Specification

### Layout Structure
- **Header:** Split-screen design with animated vinyl record
- **Hero:** Couple names with romantic tagline
- **Main Content:** 3x2 grid (desktop), 2x3 grid (tablet), 1 column (mobile)
- **Footer:** Vintage liner notes style

### Visual Design
- **Color Palette:**
  - Primary Pink: #FF9EAA (bubblegum pink)
  - Primary Blue: #5D8AA8 (air force blue)
  - Background: #FFF5F7 (pale pink)
  - Text: #2D2D2D (charcoal)
  - Accent: #FFFFFF (white borders)
- **Typography:**
  - Headings: 'Chonburi' (bold, curvy)
  - Body: 'Courier Prime' (typewriter style)
- **Visual Effects:**
  - CRT scanline overlay
  - Hard drop shadows (cut-out paper look)
  - Hover rotation on memory cards
  - Duo-tone filter on images

### Components
1. **Memory Card (x6):**
   - Polaroid-style frame
   - 4:3 aspect ratio image slot
   - Sticky note area for text
   - Hover: slight rotation + glitch effect

2. **Hero Section:**
   - Animated spinning vinyl
   - Split pink/blue background
   - Names + tagline

3. **Footer:**
   - Vintage credits style

## Functionality
- Responsive grid layout
- Image duo-tone filters
- Hover animations
- Lightbox modal for full-size view

## Acceptance Criteria
- [x] Blue and pink TV Girl aesthetic
- [x] 6 photo slots with notes area
- [x] Responsive layout
- [x] Placeholder images for future photos
- [x] Cute vintage styling
